# Idle Inspections & Repair

A modern, mobile-responsive website for Idle Inspections & Repair, a local auto mechanic business in San Angelo, TX. This site features a clean design with a "Call First" strategy, making it easy for customers to get in touch directly with the mechanic.

## Features

- Fully responsive design optimized for mobile, tablet, and desktop
- Direct call-to-action buttons throughout the site
- Services showcase with detailed descriptions
- Customer testimonials
- Integrated Google Maps for easy navigation
- Floating call button on mobile devices
- Custom color palette derived from the Idle business logo

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open your browser to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The build output will be in the `dist` directory.
